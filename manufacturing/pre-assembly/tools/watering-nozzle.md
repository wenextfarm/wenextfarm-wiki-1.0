# Step 1: Assemble the watering nozzle

Press the **watering nozzle bottom** into the **watering nozzle top**. The two pieces only fit together in one orientation.

![watering nozzle pieces connected](_images/watering_nozzle_pieces_connected.png =700px)

# Step 2: Install the basic tool hardware

Install the **magnets** and tool verification **jumper link** using [M5 x 30mm screws](M5 x 30mm screws) and **M3 x 10mm screws** and the instructions in the [basic tool hardware reference guide](../../../extras/reference/basic-tool-hardware.md). The M5 screws will secure the **watering nozzle bottom** and the **watering nozzle top** together.

![watering nozzle with basic tool hardware](_images/watering_nozzle_with_basic_tool_hardware_1.png =700px)
![watering nozzle with basic tool hardware](_images/watering_nozzle_with_basic_tool_hardware_2.png =700px)

# Step 3: Pack

Insert the **watering nozzle box insert** in the **watering nozzle box**. Then place the **pre-assembled watering nozzle** into the box and close the lid.
